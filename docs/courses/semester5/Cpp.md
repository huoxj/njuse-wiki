# C++ 高级程序设计

## 课程信息
0.2
**保研必修课**，讲 C++。内容上有两个部分。第一部分主要讲 C++ 结构化编程的特性，比如数据结构与类型、函数、泛型、模板等等；第二部分主要讲 C++ 中的面向对象程序设计，以及一些现代 C++。

- 授课老师：郑涛、潘敏学
- 课时：3 学分。每周一次课，前 2 个小时上课，后 1 个小时上机 (2024)
- 成绩：平时成绩 (40%，包括平时作业、签到、平时机考)，期末笔试 (40%)，期末机考 (20%)

- 教材：太多了，找 PPT 看吧
