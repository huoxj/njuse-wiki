# Hi there 👋

## Welcome to NJU Software Engineering Wiki!

这里是南京大学软件学院 Wiki，致力于收录南软学习相关的信息与资源，包括但不限于：

- 课程信息
- 经验分享
- 资源门户与导航

本网站旨在实现信息的自由共享，打破信息壁垒，帮助同学更好地规划课程，分享学习经验，提升教育资源的透明度和可获取性。

网站目前正处于初期搭建阶段，存在资源不完善的现象敬请谅解。同时，我们十分期待你的贡献，详见[参与贡献](#参与贡献)。
